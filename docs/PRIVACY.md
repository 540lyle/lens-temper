# Privacy Policy

LensTemper is a local, repository-distributed review workflow. The skill package
in this repository does not run a hosted service, set cookies, collect
telemetry, or transmit data to LensTemper maintainers.

Agents using LensTemper may read the plan, source files, review artifacts, and
other repository files that you explicitly point them at or that your local
agent environment makes available. Any data handling by Codex, Claude Code,
Cursor, GitHub, or another host application is governed by that application's
own settings and policies.

Do not include secrets, credentials, private keys, or regulated data in review
plans or artifacts unless your local agent environment is approved for that
data.
